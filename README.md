#### Keymaps

|Key|Info|
|:--:|:--:|
|H|10h|
|J|10j|
|K|10k|
|L|10l|
|Ctrl t|Open Nvim-Tree|
|Tab|Select next with nvim-cmp|
|Shift + Tab|Select pervious with nvim-cmp|
|Enter|Select highlight item with nvim-cmp|
|Enter|Open file in Nvim-Tree|
